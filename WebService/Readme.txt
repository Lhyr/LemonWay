This WebService is decomposed into 2 classes:

SwFibonacci.asmx which contains a method to calculate Fibonacci sequence.
swXmlToJson.asmx which can convert Xml to Json using NewtonSoft Library.

To call this web service, you need to have IIS installed.

The address is by default http://localhost/WebService, followed by SwFibonacci.asmx or swXmlToJson.asmx depending what you need.
